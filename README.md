# JPTRIPGRAM

## 프로젝트 소개
일본 여행 경험을 기록·공유하는 SNS형 여행 블로그 웹 서비스

## 기술 스택
- Java 17
- JSP / Servlet / JDBC
- Oracle DB
- Tomcat 9
- HTML / CSS / JavaScript

## 주요 기능
- 회원가입 / 로그인 / 로그아웃
- 게시글 CRUD
- 댓글 / 좋아요
- 마이페이지

## 아키텍처
- MVC 패턴 (Controller / Service / DAO / View)

## 실행 방법
1. Tomcat 9 설치
2. Oracle DB 세팅
3. SQL 실행
4. Eclipse에서 Import → Run
